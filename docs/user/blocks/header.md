# Header Block

The header block displays a header with an optional icon and title.

This block is static and does not award points.

When placed as the first block on a page, the header block replaces the default page header and icon. I designed the header block with a few use cases in mind:

- Page headers
- Decorative elements (icons only)
- Section headers

## Options

When creating a header block, you can configure:

- **Icon:** Optional icon from the [Lucide icon library](https://lucide.dev/icons/). Enter just the icon name (e.g., `star`, `trophy`, `map-pin`)
- **Title Text:** The header text to display
- **Title Size:** Either `Small`, `Medium`, or `Large` to adjust the prominence of the title
